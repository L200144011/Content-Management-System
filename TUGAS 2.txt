Modules:

Administration Menu
https://www.drupal.org/project/admin_menu
Menggantikan overlay menu dengan dropdown menu yang akan mempersingkat navigasi pengaturan Drupal

Coffee
https://www.drupal.org/project/coffee
Hotkey untuk menampilkan kolom akses menu administrasi Drupal secara cepat

Jquery Update
https://www.drupal.org/project/jquery_update
(Tidak wajib) Memperbaharui jQuery yang telah ada dalam inti Drupal dengan versi terbaru jQuery

Bootstrap Library
https://www.drupal.org/project/bootstrap_library
(Tidak wajib) Mengatur kode sumber Bootstrap

CKEditor
https://www.drupal.org/project/ckeditor
Fitur edit visual HTML untuk Drupal

IMCE
https://www.drupal.org/project/imce
Fitur dialog untuk mengunggah dan mengatur file di Drupal

Empty Front Page
https://www.drupal.org/project/empty_front_page
Mengubah cara kerja Drupal untuk menampilkan blok saja, dari yang sebelumnya menampilkan konten

Special menu items
https://www.drupal.org/project/special_menu_items
Modul yang akan memberi "kolom semu" dan pembatas pada menu item. Bentuk kolom semu mirip seperti menu tapi bukan merupakan link, sedangkan pembatas berbentuk seperti "-------" yang fungsi utamanya untuk membatasi dan mengatur menu

PathAuto
https://www.drupal.org/project/pathauto
Modul yang akan secara otomatis membuat URL/alur singkat untuk berbagai konten, seperti node, tax, user, tanpa campur tangan pengguna ecara manual. URL yang diciptakan berupa /kategori/node-judul tidak seperti URL yang biasa. Pengguna dapat mengatur dasar alur yang akan digunakan untuk membuat alur secara otomatis nantinya

Token
https://www.drupal.org/project/token
(Dibutuhkan oleh PathAuto) Menyediakan token tambahan yang biasanya tidak ada, atau tidak didukung oleh inti Drupal, juga berfungsi sebagai bagan pengaturan token

Module Filter
https://www.drupal.org/project/module_filter
Modul untuk mencari, menemukan, menghidupkan atan mematikan serta mengatur modul Drupal

Views
https://www.drupal.org/project/views
Menciptakan gambaran Drupal

Chaos tool suite (ctools)
https://www.drupal.org/project/ctools
(Dibutuhkan oleh Views) Modul yang memiliki sepaket API dan perkakas tambahan untuk menambah pengalaman pengembang, juga memiliki modul Page Manager yang dapat digunakan untuk mengatur halaman

Views Bootstrap
https://www.drupal.org/project/views_bootstrap
Modul untuk menciptakan komponen tema Bootstrap dengan fitur modul Views

DraggableViews
https://www.drupal.org/project/draggableviews
Modul yang memberi kebebasan untuk "menarik" baris tampilan, atau yang biasa orang katakan "drag n drop" namun di sini digunakan untuk mengatur baris tampilan

Entity API
https://www.drupal.org/project/entity
Modul yang memberi entitas API tambahan untuk inti Drupal yang dapat digunakan untuk menyediakan jalan bagi entitas dan atributnya, serta memiliki tambahan untuk entitas kendali CRUD yang dapat membantu menyingkat pembuatan jenis entitas baru

CKEditor Link - A plugin to easily create links to Drupal internal paths
https://www.drupal.org/project/ckeditor_link
Modul yang dapat memudahkan dalam menciptakan link ke inti Drupal melewati tombol Link dan dialog

Block Class
https://www.drupal.org/project/block_class
Modul yang dapat memudahkan pengguna untuk menambah kelas untuk blok manapun

Plupload integration
https://www.drupal.org/project/plupload
Modul untuk memudahkan pengguna dalam mengunggah beberapa berkas gambar

Devel
https://www.drupal.org/project/devel
Modul yang menyediakan fungsi pembantu untuk pengembang Drupal

Image URL Formatter
https://www.drupal.org/project/image_url_formatter
Modul yang dapat menambah fitur format URL dalam kolom gambar, sangat memudahkan dalam mengunggah gambar yang sebelumnya ada di website lain hanya dengan mencantumkan URL gambar tersebut

Equal Heights jQuery
https://www.drupal.org/project/equalheights
Modul yang akan mengimplementasikan plugin jQuery Equal Heights yang akan mengatur tingginya elemen dalam kelas tententu

Libraries API
https://www.drupal.org/project/libraries
(Dibutuhkan oleh jQuery Equal Heights) Modul denominator untuk semua modul/tema/profil Drupal yang terintegrasi dengan pustaka luar
